# yusa
```yusa``` is support tool for [tomori](https://github.com/koike/tomori) and [ayumi](https://github.com/koike/ayumi)

## Require
- PHP 7
- Composer

## Install
```sh
# clone repository (in the same hierarchy as tomori and ayumi)
$ git clone https://github.com/koike/yusa.git

# write api key
$ vi .env

# setup files
$ php install.php
```
